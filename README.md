# shoppingMall

#### 介绍
基于SSM实现的购物商城管理系统

#### 软件架构
Spring+SpringMVC+Myatis+Layui+jsp

#### 功能模块介绍
##### 用户端模块：
1.  注册登录，使用cookie记住密码
2.  购物车管理
3.  上传下载文件
##### 后台管理员端模块：
1.  商品管理
2.  用户管理
3.  订单管理

#### 使用说明
1.  新建mysql数据库product，建表及数据初始化语句见resources/product.sql
2.  启动项目后地址栏访问：http://localhost:8080/LayuiShop/，即可访问项目
